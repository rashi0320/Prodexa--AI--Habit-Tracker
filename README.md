#  Prodexa – AI-Powered Habit Tracker

Prodexa is a full-stack AI-driven habit tracking web application designed to help users build consistency, stay motivated, and improve daily productivity through data-driven insights and reward-based reinforcement.

---

##  Features

-  User Authentication (Signup/Login/Logout)
-  Create, Complete & Delete Daily Tasks
-  Reward System (Earn points for completed habits)
-  Consistency Tracking (Performance analytics)
-  AI-based Habit Insights (Behavior tracking & analysis)
-  Daily Reminder System (via scheduler)
-  Personal Dashboard with task overview

---

##  Motivation

I built Prodexa to solve a common problem — maintaining consistency in daily habits. Most people start strong but struggle to continue. This app uses **AI concepts and data analysis** to track behavior patterns and encourage users through rewards and insights.

##  How It Works

1. User signs up and logs in
2. Adds daily habits/tasks
3. Marks tasks as completed
4. Earns reward points for consistency
5. Tracks performance via dashboard
6. Scheduler checks for incomplete tasks daily and sends reminders (console-based)
